A set of customized Astro Starlight UI components

## Prerequisites

You will need to have an Astro or Astro + Starlight website set up.
If you don't have one yet, you can follow the ["Getting Started"](https://starlight.astro.build/getting-started) guide in the Starlight docs to create one.

## Installation

import { Steps, Tabs, TabItem } from '@astrojs/starlight/components'

<Steps>

1. Install `astro-nebula-ui` as an npm package by running the following command in your terminal:

   <Tabs syncKey="pkg">

   <TabItem label="npm">

   ```sh
   npm install astro-nebula-ui
   ```

   </TabItem>

   <TabItem label="pnpm">

   ```sh
   pnpm add astro-nebula-ui
   ```

   </TabItem>

   <TabItem label="Yarn">

   ```sh
   yarn add astro-nebula-ui
   ```

   </TabItem>

   </Tabs>

2. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the components that you added in action.

</Steps>
