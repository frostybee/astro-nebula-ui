import { CheatSheet, CheatSheetSection, CheatSheetResource, CheatSheetResources, CodeBlock } from 'astro-nebula-ui';

The CheatSheet component system provides a structured way to create reference cards that are perfect for educational content. Each cheatsheet can contain multiple sections with code examples, explanations, and a dedicated resources section.

## Basic Usage

Here's a simple cheatsheet with multiple sections:

<CheatSheet title="CSS Flexbox Cheatsheet" description="Essential flexbox properties and values">
  <CheatSheetSection title="Container Properties" icon="📦">
    <CodeBlock lang="css" code="display: flex;" />
    <p>Makes an element a flex container, enabling flexbox layout for its children.</p>
    
    <CodeBlock lang="css" code="flex-direction: row | column | row-reverse | column-reverse;" />
    <p>Defines the main axis direction of the flex items.</p>
  </CheatSheetSection>

  <CheatSheetSection title="Item Properties" icon="🔧">
    <CodeBlock lang="css" code="flex: 1;" />
    <p>Shorthand for flex-grow, flex-shrink, and flex-basis.</p>
    
    <CodeBlock lang="css" code="align-self: flex-start | flex-end | center | stretch;" />
    <p>Overrides the align-items value for individual flex items.</p>
  </CheatSheetSection>

  <CheatSheetResources slot="resources">
    <CheatSheetResource 
      title="CSS Tricks Complete Guide" 
      url="https://css-tricks.com/snippets/css/a-guide-to-flexbox/"
      description="Comprehensive guide to CSS Flexbox with visual examples"
      type="tutorial"
    />

      title="Flexbox Froggy"

      type="tool"
    />
  </CheatSheetResources>
</CheatSheet>

## Component APIs

### CheatSheet Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | Required | The main title of the cheatsheet |
| `description` | `string` | `undefined` | Optional description shown below the title |
| `variant` | `"default" \| "outline" \| "minimal" \| "compact"` | `"default"` | Visual style variant |
| `printable` | `boolean` | `true` | Show print button and enable print styles |
| `collapsible` | `boolean` | `false` | Enable collapse/expand functionality |
| `id` | `string` | auto-generated | Custom ID for the component |

### CheatSheetSection Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | Required | The section title |
| `icon` | `string` | `undefined` | Optional emoji or text icon |
| `collapsible` | `boolean` | `false` | Make this section collapsible |
| `defaultCollapsed` | `boolean` | `false` | Start in collapsed state |
| `id` | `string` | auto-generated | Custom ID for the section |

### CheatSheetResource Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | Required | The resource title |
| `url` | `string` | Required | The resource URL |
| `description` | `string` | `undefined` | Optional description of the resource |
| `type` | Resource type | `"reference"` | Type of resource (affects icon and styling) |
| `external` | `boolean` | `true` | Whether the link opens in a new tab |

#### Resource Types

- `"documentation"` - Official docs, API references
- `"tutorial"` - Step-by-step guides and tutorials  
- `"tool"` - Interactive tools and utilities
- `"reference"` - Quick reference materials
- `"video"` - Video content and courses
- `"article"` - Blog posts and articles
- `"book"` - Books and comprehensive guides
- `"course"` - Structured learning courses

### CheatSheetResources Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"Resources & References"` | The resources section title |
| `collapsible` | `boolean` | `false` | Make the resources section collapsible |
| `defaultCollapsed` | `boolean` | `false` | Start in collapsed state |
| `id` | `string` | auto-generated | Custom ID for the section |

## Best Practices

1. **Keep sections focused** - Each section should cover one specific topic
2. **Use descriptive titles** - Make it easy to scan and find information
3. **Include practical examples** - Show real-world usage, not just syntax
4. **Organize logically** - Structure content from basic to advanced concepts
5. **Provide quality resources** - Link to official docs and trusted tutorials
6. **Use appropriate icons** - Visual cues help with quick recognition
7. **Consider print layout** - Test how your cheatsheet looks when printed
8. **Make it scannable** - Use headings, code blocks, and short paragraphs

## Integration with Other Components

CheatSheet components work seamlessly with other components in your library:

- **CodeBlock**: Automatically integrates with copy functionality
- **Callouts**: Use inside sections for important notes
- **CollapsiblePanel**: Similar collapsible behavior patterns
- **LearningObjectives**: Use together for complete lesson structure

This component system provides a powerful way to create comprehensive reference materials that enhance the learning experience for students.
