# NewCallout Component Examples

The `NewCallout` component is an enhanced version of the basic callout with additional features like collapsible content, multiple variants, custom titles, and improved Starlight theme integration.

## Collapsible Callouts

### Expanded by Default

<NeoCallout type="info" title="Click to collapse this section" collapsible={true} expanded={true}>
This callout starts expanded but can be collapsed by clicking the header. Perfect for optional reading material or detailed explanations that might clutter the main content.

You can include:
- Lists and bullet points
- **Bold text** and *italic text*
- [External links](https://example.com)
- `Code snippets` and blocks
- Multiple paragraphs with detailed explanations
</NeoCallout>

<NeoCallout type="warning" title="Advanced Configuration (Click to expand)" collapsible={true} expanded={false}>

### API Reference

<NeoCallout type="info" title="Component Props" collapsible={true} expanded={false}>

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `string` | `"info"` | Callout type: info, note, danger, success, warning, tip, quote |
| `title` | `string` | `undefined` | Optional title displayed in the header |
| `icon` | `string` | `undefined` | Custom SVG icon (overrides default type icon) |
| `variant` | `string` | `"default"` | Visual variant: default, minimal, filled, outline |
| `collapsible` | `boolean` | `false` | Whether the callout can be collapsed |
| `expanded` | `boolean` | `true` | Initial expanded state (only applies if collapsible) |

</NeoCallout>

## Best Practices

<NeoCallout type="tip" title="Usage Guidelines">
- Use **info** callouts for supplementary information
- Reserve **danger** callouts for critical warnings only
- Keep callout content concise and focused
- Use collapsible callouts for optional or detailed content
- Choose variants that match your design system
</NeoCallout>

<NeoCallout type="warning" title="Accessibility Considerations">
- Always provide meaningful titles for collapsible callouts
- Ensure sufficient color contrast in custom themes
- Test keyboard navigation with collapsible content
- Use semantic HTML structure within callout content
</NeoCallout>
